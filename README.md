# # Husseinyouth Portfolio

موقع تعريفي لمؤسسة شباب الحسين للمساعدات الإنسانية وتطبيقاتها.

## التطبيقات
- حقيبة الكوثر
- بنك الدم العراقي
- سفراء الولاية

تم تطوير الموقع باستخدام Flutter Web.
